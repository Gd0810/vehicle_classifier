About This Project

🧠

Architecture

ResNet-50 pre-trained on ImageNet with a custom classification head: Dropout → Linear(2048→512) → ReLU → Dropout → Linear(512→21) → Sigmoid

📊

Dataset

Multi-label classification dataset with 3,615+ vehicle images across 21 classes. CSV-based label mapping with one-hot encoding.

⚙️

Training Strategy

Transfer learning with frozen backbone layers. Only layer4 and the custom head are fine-tuned using Adam optimizer + StepLR scheduler.

🎯

Loss Function

Binary Cross-Entropy Loss (BCELoss) for multi-label classification. Threshold 0.5 is used to binarize predictions.

🚀

Quick Start

pip install -r requirements.txt
# Add images to ./images/
python train.py --images_dir ./images
python app.py
